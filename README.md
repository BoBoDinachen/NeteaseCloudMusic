
## NeteaseCloudMusic 网易云音乐WebApp 开发
> 仿网易云客户端的在线WebAPP
> * 作者: 薛定谔的猫猫虫 😎
> * 技术栈: React+TypeScript+Vite+TailwindCSS ✨
> * 描述🔧: 后端采用github开源项目--NeteaseCloudMusicApi，提供api服务
> * 目标🧡：一个寒假内完成，功能点---1.用户模块 2.音乐搜索模块 3.歌单模块 4.音乐播放器 5.排行榜
